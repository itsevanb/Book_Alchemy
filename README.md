# Book Library Flask Application

This application is a simple Flask web app that allows users to manage a library of books.

## Features

- Display a list of books with their corresponding details.
- Sort books by title or author's name.
- Search books by title or author's name.
- Add new books by entering details manually.
- Add new authors by entering their details manually.
- Delete books from the library.
-Stored books in a SQLalchemy database.

## Installation and Setup

1. Clone the repository

2. Change into the directory

3. Install the required packages

4. Set up the database

5. Run the application

6. Visit `http://localhost:5000` in your web browser to view the application.

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Bootstrap
- Open Library API
